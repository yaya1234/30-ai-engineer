# Day2: NumPy/Pandas 高级操作与 Titanic EDA

## 文件说明
- `numpy_practice.py`：NumPy 基础练习（数组创建、切片、广播、矩阵运算等）
- `pandas_advanced.py`：Pandas 高级操作（分组、透视表、时间序列、apply）
- `analysis_report.py`：完整数据分析报告，生成可视化图表和清洗后数据

## 运行环境
- Python 3.10+
- 安装依赖：`pip install numpy pandas matplotlib seaborn`

## 运行方法
```bash
python numpy_practice.py
python pandas_advanced.py
python analysis_report.py
